# Images du site

## Capture d'écran du formulaire de devis

Pour ajouter la capture d'écran du formulaire :

1. **Nom du fichier** : `formulaire-devis-capture.png`
2. **Emplacement** : Ce dossier (`public/images/`)
3. **Format recommandé** : PNG ou WebP
4. **Taille recommandée** : 1920x1200px (ratio 16:10)

Le code dans `src/app/page.tsx` est déjà configuré pour afficher cette image automatiquement une fois qu'elle est placée ici.

## Description de la capture attendue

La capture d'écran doit montrer :
- Le formulaire de configuration (colonne gauche)
- Les 6 formules de devis comparables (colonne droite)
- Le détail des prix et options
